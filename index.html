Pjs.

Pjs is a simple Processing.js IDE built using node-webkit, Angular.js and ACE editor. It's still in pre-alpha-beta-whatever, so don't expect many features (Including File IO). At the moment it will render valid Processing.JS code, however.

Give it a shot if you feel brave,

$ git clone https://github.com/mrkev/pjs
$ cd pjs
$ npm install
$ gulp run
